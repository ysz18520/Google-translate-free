## 快速开始

### 1. 安装应用

#### 方式一：直接授权安装（推荐）

将以下链接发送给店铺 Owner，对方点击后按提示完成授权：

```
https://admin.coollaa.com/api/shopify/auth?shop=你的店铺.myshopify.com
```

步骤：
1. 点击链接 → 登录 Shopify 账号
2. 查看应用请求的权限 → 点击**安装应用**
3. 授权成功后自动跳转到管理后台

#### 方式二：从 Shopify App Store 安装

如应用已上架，直接在 Shopify App Store 搜索 **AutoTranslate Pro** 安装。

---

### 2. 店铺前台使用

安装完成后，访问店铺前台页面，右下角会出现语言切换器：

#### 切换语言

1. 点击右下角**地球图标**悬浮按钮
2. 在弹出的下拉菜单中选择目标语言：
   - 🇺🇸 English（英语）
   - 🇪🇸 Español（西班牙语）
   - 🇫🇷 Français（法语）
   - 🇸🇦 العربية（阿拉伯语）
   - 🇨🇳 中文
3. 页面文字自动替换为目标语言

#### 首次访问 vs 切换语言

| 场景 | 体验 |
|------|------|
| **首次访问（外部进入）** | 显示网站原文语言，不自动翻译 |
| **手动切换语言** | Google Translate 实时翻译，约 1-2 秒完成 |
| **再次切换同一语言** | Google 自动缓存，页面秒开 |

#### 网络注意事项

由于使用 Google Translate 免费组件，**国内用户需开启 VPN** 以确保翻译功能正常加载。国外用户可直接使用，无需额外配置。

#### RTL 语言支持

切换阿拉伯语（العربية）时，页面自动调整为**从右到左（RTL）**布局。

### 3. 管理后台使用

在 Shopify 后台 → 应用 → **AutoTranslate Pro** 进入管理界面。

#### 概览页

- 查看已授权店铺信息
- 查看支持的语言列表
- 产品功能介绍和使用说明

#### 店铺调试模式

1. 在概览页点击 **"进入店铺调试模式"**
2. 系统生成调试链接 → 点击链接跳转到店铺前台
3. 该浏览器获得永久调试授权
4. 在店铺前台按 `Ctrl + Shift + L` 弹出调试菜单：
   - **退出调试模式**：清除当前浏览器授权

#### 设置页

配置前端语言切换器的外观：

| 配置项 | 说明 | 选项 |
|--------|------|------|
| **悬浮按钮位置** | 语言切换器显示在页面哪个角落 | 右下角 / 左下角 / 隐藏 |
| **主题色** | 悬浮按钮和菜单的主题颜色 | 任意颜色 |
| **启用语言** | 店铺支持哪些语言 | 多选：英/西/法/阿/中 |
| **网站原文语言** | 自动检测（从 `<html lang>` 读取） | 只读显示，不可手动配置 |

### 4. 常见问题排查

#### 店铺前台看不到语言切换器

| 排查项 | 操作 |
|--------|------|
| 应用是否安装成功 | 在 Shopify 后台 → 应用 中确认 |
| 是否完成授权 | 重新访问授权链接完成 OAuth |
| 浏览器控制台报错 | 按 F12 → Console 查看红色报错 |
| 脚本是否注入 | 查看页面源码中是否包含 `translator-widget.js` |

#### 切换语言后页面没变化

1. 等待 1-2 秒，Google Translate 实时翻译需要加载
2. 检查是否能正常访问 `translate.google.com`（国内需 VPN）
3. 按 `Ctrl + Shift + R` 强制刷新页面

#### 翻译内容不更新

1. Google Translate 会自动更新翻译内容，无需手动刷新
2. 如遇缓存问题，切换回源语言后再重新选择目标语言
3. 强制刷新页面 `Ctrl + Shift + R`

## 功能介绍

### 多语言翻译引擎

- **翻译引擎**：Google Translate 免费网页组件（`element.js`）
- **支持语言**：英语（en）、西班牙语（es）、法语（fr）、阿拉伯语（ara）、中文（zh），可通过自定义语言扩展至 100+ 种
- **成本**：零成本，无需 API Key
- **工作原理**：加载 Google 翻译组件，用户选择语言后自动抓取并重写页面 HTML

### 前端语言切换器

- **悬浮按钮**：固定在页面右下角、左下角或隐藏，不影响页面布局
- **下拉菜单**：点击后展开语言选择列表，带国旗图标
- **套壳 UI**：隐藏 Google 默认横幅和下拉框，使用品牌一致的自定义界面
- **响应式适配**：自动适配 PC 端和移动端屏幕
- **主题定制**：支持自定义按钮颜色和位置
- **语言持久化**：localStorage 记住用户选择，刷新页面后保持

### 店主调试工具

- **快捷键触发**：`Ctrl + Shift + L`（仅限已授权浏览器）
- **权限管理**：通过管理后台生成调试链接授权，永久有效

### 管理后台

嵌入在 Shopify Admin 内的管理界面：

- **概览**：店铺信息、产品介绍、使用说明
- **设置**：语言切换器外观配置（位置、颜色、启用语言、自定义语言）

### 技术特性

| 特性 | 说明 |
|------|------|
| **OAuth 2.0 授权** | 标准 Shopify 授权流程 |
| **ScriptTag 注入** | 授权后自动将翻译脚本注入店铺主题 |
| **RTL 支持** | 阿拉伯语自动启用从右到左布局 |
| **数据隔离** | 每个店铺的设置数据独立存储 |
| **安全设计** | 所有凭证通过环境变量管理，禁止硬编码 |

## 技术栈

- **后端**：Node.js 20 + Express + Prisma ORM
- **数据库**：MySQL 8
- **翻译引擎**：Google Translate 免费网页组件（前端实时翻译）
- **前端管理**：纯 HTML/CSS/JS（嵌入 Shopify Admin）
- **前端店铺**：注入脚本（纯 JS，无依赖）
- **部署**：Docker + Docker Compose + Nginx

## 与 Shopify Translate & Adapt 官方插件的对比

AutoTranslate Pro 与 Shopify 官方多语言方案在设计理念和技术实现上有本质区别：

| 对比项 | AutoTranslate Pro | Shopify Translate & Adapt |
|--------|-------------------|---------------------------|
| **技术路线** | 前端 Google Translate 组件（ScriptTag 注入 JS） | 服务端 Markets 多市场渲染 |
| **URL 路径** | 不变化，同一页面动态切换语言 | 自动生成 `/es`、`/zh` 等子路径 |
| **语言切换方式** | 悬浮按钮 + 手动切换 | 路径跳转 + 市场绑定 |
| **翻译存储** | Google 实时翻译（无本地存储） | Shopify 原生存储 |
| **SEO** | 依赖原始页面 SEO，多语言不额外生成 URL | 每个语言有独立 URL，利于多语言 SEO |
| **适用场景** | 中小型店铺，快速上线多语言，零成本 | 大型店铺，需要完整多语言 SEO |
| **成本** | **零成本**（使用 Google 免费组件） | Shopify Markets 功能可能需要更高套餐 |

### 各自的优劣

**AutoTranslate Pro 优势：**
- 无需修改 Shopify 主题或配置 Markets
- 安装即可用，5 分钟完成配置
- **零成本**（使用 Google Translate 免费组件）
- 支持 100+ 种语言（通过自定义语言扩展）
- 源语言自动检测（从网站 `<html lang>` 读取）
- 翻译质量由 Google 提供，持续更新优化

**AutoTranslate Pro 局限：**
- 不生成多语言独立 URL，对多语言 SEO 辅助有限
- 需要加载 Google Translate 脚本，国内用户需 VPN
- 极复杂主题可能存在少量文本翻译遗漏

**Shopify 官方优势：**
- 完整的多语言 SEO 支持（独立 URL、hreflang 标签）
- 服务端渲染，无前端替换延迟
- 与 Shopify 生态系统深度集成

**建议：** 如果店铺对多语言 SEO 要求极高（如需要 Google 多语言收录），建议同时使用 Shopify Markets；如果追求快速、低成本、易维护的多语言方案，AutoTranslate Pro 是更优选择。
