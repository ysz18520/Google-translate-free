## 快速开始

### 1. 安装应用

#### 方式一：直接授权安装（推荐）

将以下链接发送给店铺 Owner，对方点击后按提示完成授权：

```
https://admin.coollaa.com/api/shopify/auth?shop=你的店铺.myshopify.com
```

步骤：
1. 点击链接 → 登录 Shopify 账号
2. 查看应用请求的权限 → 点击**安装应用**
3. 授权成功后自动跳转到管理后台

#### 方式二：从 Shopify App Store 安装

如应用已上架，直接在 Shopify App Store 搜索 **AutoTranslate Pro** 安装。

---

### 2. 店铺前台使用

安装完成后，访问店铺前台页面，右下角会出现语言切换器：

#### 切换语言

1. 点击右下角**地球图标**悬浮按钮
2. 在弹出的下拉菜单中选择目标语言：
   - 🇺🇸 English（英语）
   - 🇪🇸 Español（西班牙语）
   - 🇫🇷 Français（法语）
   - 🇸🇦 العربية（阿拉伯语）
   - 🇨🇳 中文
3. 页面文字自动替换为目标语言

#### 首次访问 vs 切换语言

| 场景 | 体验 |
|------|------|
| **首次访问（外部进入）** | 显示网站原文语言，不自动翻译 |
| **手动切换语言** | 页面文字实时翻译，约 2-3 秒完成 |
| **再次切换同一语言** | 直接读取缓存，页面秒开（0.1 秒） |

#### RTL 语言支持

切换阿拉伯语（العربية）时，页面自动调整为**从右到左（RTL）**布局。

### 3. 管理后台使用

在 Shopify 后台 → 应用 → **AutoTranslate Pro** 进入管理界面。

#### 概览页

- 查看已授权店铺信息
- 查看翻译统计（已翻译商品数、页面数、API 调用次数）
- 查看支持的语言列表

#### 店铺调试模式

> 用于店主在前端直接刷新翻译缓存，无需登录服务器。

1. 在概览页点击 **"进入店铺调试模式"**
2. 系统生成调试链接 → 点击链接跳转到店铺前台
3. 该浏览器获得永久调试授权
4. 在店铺前台按 `Ctrl + Shift + L` 弹出调试菜单：
   - **刷新当前页翻译**：重新翻译当前浏览的页面
   - **刷新全站翻译**：标记全站翻译缓存为待更新
   - **退出调试模式**：清除当前浏览器授权

#### 商品翻译

1. 切换到**商品翻译**标签
2. 浏览店铺商品列表
3. 选择需要翻译的商品 → 点击**批量翻译**
4. 系统自动翻译商品标题和描述

#### 页面翻译

1. 切换到**页面翻译**标签
2. 浏览店铺页面列表（About Us、Contact 等）
3. 选择需要翻译的页面 → 点击**批量翻译**

#### 设置页

配置前端语言切换器的外观：

| 配置项 | 说明 | 选项 |
|--------|------|------|
| **悬浮按钮位置** | 语言切换器显示在页面哪个角落 | 右下角 / 左下角 |
| **主题色** | 悬浮按钮和菜单的主题颜色 | 任意颜色 |
| **启用语言** | 店铺支持哪些语言 | 多选：英/西/法/阿/中 |
| **网站原文语言** | 自动检测（从 `<html lang>` 读取） | 只读显示，不可手动配置 |

### 4. 常见问题排查

#### 店铺前台看不到语言切换器

| 排查项 | 操作 |
|--------|------|
| 应用是否安装成功 | 在 Shopify 后台 → 应用 中确认 |
| 是否完成授权 | 重新访问授权链接完成 OAuth |
| 浏览器控制台报错 | 按 F12 → Console 查看红色报错 |
| 脚本是否注入 | 查看页面源码中是否包含 `translator-widget.js` |

#### 切换语言后页面没变化

1. 首次翻译需要 2-3 秒，请等待
2. 检查百度翻译 API 额度是否用完
3. 在管理后台点击"进入店铺调试模式" → 刷新当前页翻译

#### 翻译内容不更新

1. 修改商品/页面后，Shopify 会自动通知插件更新
2. 如需立即更新：
   - 方式一：在店铺前台按 `Ctrl + Shift + L` → 刷新当前页
   - 方式二：在管理后台使用调试模式 → 刷新全站翻译

## 功能介绍

### 多语言翻译引擎

- **翻译 API**：百度翻译 API
- **支持语言**：英语（en）、西班牙语（es）、法语（fr）、阿拉伯语（ara）、中文（zh）
- **可扩展性**：后台配置可随时添加/移除语言

### 前端语言切换器

- **悬浮按钮**：固定在页面右下角或左下角，不影响页面布局
- **下拉菜单**：点击后展开语言选择列表，带国旗图标
- **响应式适配**：自动适配 PC 端和移动端屏幕
- **主题定制**：支持自定义按钮颜色和位置

### 翻译缓存系统

- **自动缓存**：首次翻译后结果存入数据库，后续直接读取
- **哈希检测**：自动检测原文是否变更，变更后自动重新翻译
- **状态管理**：
  - `translated`：缓存有效，直接返回
  - `outdated`：原文已变更，下次访问自动重新翻译
  - `pending`：正在翻译中

### Webhook 自动更新检测

安装后自动向 Shopify 注册 Webhook，实时检测内容变更：

| Webhook 事件 | 触发行为 |
|-------------|---------|
| `products/update` | 商品修改 → 标记该商品翻译为 outdated |
| `pages/update` | 页面修改 → 标记该页面翻译为 outdated |
| `articles/update` | 博客文章修改 → 标记该文章翻译为 outdated |
| `themes/update` | 主题升级 → 标记**全站**翻译为 outdated |

### 店主调试工具

- **快捷键触发**：`Ctrl + Shift + L`（仅限已授权浏览器）
- **当前页刷新**：仅重新翻译当前浏览的页面
- **全站刷新**：标记全站所有翻译为 outdated
- **权限管理**：通过管理后台生成调试链接授权，永久有效

### 管理后台

嵌入在 Shopify Admin 内的管理界面：

- **概览**：店铺信息、翻译统计、语言列表
- **商品翻译**：浏览商品列表，批量翻译
- **页面翻译**：浏览页面列表，批量翻译
- **设置**：语言切换器外观配置

### 技术特性

| 特性 | 说明 |
|------|------|
| **OAuth 2.0 授权** | 标准 Shopify 授权流程，HMAC 签名校验 |
| **ScriptTag 注入** | 授权后自动将翻译脚本注入店铺主题 |
| **RTL 支持** | 阿拉伯语自动启用从右到左布局 |
| **数据隔离** | 每个店铺的翻译数据独立存储 |
| **安全设计** | 所有凭证通过环境变量管理，禁止硬编码 |

## 技术栈

- **后端**：Node.js 20 + Express + Prisma ORM
- **数据库**：MySQL 8
- **翻译 API**：百度翻译 API
- **前端管理**：纯 HTML/CSS/JS（嵌入 Shopify Admin）
- **前端店铺**：注入脚本（纯 JS，无依赖）
- **部署**：Docker + Docker Compose + Nginx

## 与 Shopify Translate & Adapt 官方插件的对比

AutoTranslate Pro 与 Shopify 官方多语言方案在设计理念和技术实现上有本质区别：

| 对比项 | AutoTranslate Pro | Shopify Translate & Adapt |
|--------|-------------------|---------------------------|
| **技术路线** | 前端脚本动态替换（ScriptTag 注入 JS） | 服务端 Markets 多市场渲染 |
| **URL 路径** | 不变化，同一页面动态切换语言 | 自动生成 `/es`、`/zh` 等子路径 |
| **语言切换方式** | 悬浮按钮 + 浏览器自动检测 + 手动切换 | 路径跳转 + 市场绑定 |
| **翻译存储** | 数据库存储 + 内容哈希检测变更 | Shopify 原生存储 |
| **SEO** | 依赖原始页面 SEO，多语言不额外生成 URL | 每个语言有独立 URL，利于多语言 SEO |
| **适用场景** | 中小型店铺，快速上线多语言 | 大型店铺，需要完整多语言 SEO |
| **成本** | 仅需百度翻译 API 费用 | Shopify Markets 功能可能需要更高套餐 |

### 各自的优劣

**AutoTranslate Pro 优势：**
- 无需修改 Shopify 主题或配置 Markets
- 安装即可用，5 分钟完成配置
- 源语言自动检测（从网站 `<html lang>` 读取）
- 智能增量翻译，只更新变更内容
- 店主可随时在应用界面一键刷新全站翻译

**AutoTranslate Pro 局限：**
- 不生成多语言独立 URL，对多语言 SEO 辅助有限
- 首次访问需要加载翻译脚本，有轻微延迟
- 极复杂主题可能存在文本替换遗漏

**Shopify 官方优势：**
- 完整的多语言 SEO 支持（独立 URL、hreflang 标签）
- 服务端渲染，无前端替换延迟
- 与 Shopify 生态系统深度集成

**建议：** 如果店铺对多语言 SEO 要求极高（如需要 Google 多语言收录），建议同时使用 Shopify Markets；如果追求快速、低成本、易维护的多语言方案，AutoTranslate Pro 是更优选择。
