# 操作手册：多渠道营销活动

> **模式**：NEXUS-Micro 到 NEXUS-Sprint | **时长**：2-4 周 | **智能体**：10-15

---

## 场景

你要在多个渠道同时搞一个营销活动。内容要针对各平台定制，品牌要一致，决策要基于数据。活动需要带来可衡量的获客和互动。

## 智能体阵容

### 活动核心
| 智能体 | 角色 |
|--------|------|
| 社交媒体策略师 | 活动负责人，跨平台策略 |
| 内容创作者 | 各种格式的内容产出 |
| 增长黑客 | 获客策略，漏斗优化 |
| 品牌守护者 | 跨渠道品牌一致性 |
| 数据分析师 | 效果追踪和优化 |

### 平台专家
| 智能体 | 角色 |
|--------|------|
| Twitter 互动官 | Twitter/X 活动执行 |
| TikTok 策略师 | TikTok 内容和增长 |
| Instagram 策展人 | Instagram 视觉内容 |
| Reddit 社区建设者 | Reddit 真实互动 |
| 应用商店优化师 | 应用商店（如果是移动端） |

### 支持
| 智能体 | 角色 |
|--------|------|
| 趋势研究员 | 市场时机和趋势对齐 |
| 实验追踪员 | A/B 测试活动变体 |
| 高管摘要生成器 | 活动汇报 |
| 法务合规员 | 广告合规，信息披露要求 |

## 执行计划

### 第 1 周：策略与内容制作

```
第 1-2 天：活动策略
├── 社交媒体策略师 → 跨平台活动策略
│   ├── 活动目标和 KPI
│   ├── 目标受众定义
│   ├── 平台选择和预算分配
│   ├── 内容日历（4 周计划）
│   └── 各平台互动策略
│
├── 趋势研究员 → 市场时机分析
│   ├── 可以蹭的热点话题
│   ├── 竞品活动分析
│   └── 最佳发布时机
│
├── 增长黑客 → 获客漏斗设计
│   ├── 着陆页优化方案
│   ├── 转化漏斗图
│   ├── 病毒式传播机制（推荐、分享）
│   └── 渠道预算分配
│
├── 品牌守护者 → 活动品牌规范
│   ├── 活动专用视觉规范
│   ├── 话术框架
│   ├── 活动语调
│   └── 该做和不该做的
│
└── 法务合规员 → 广告合规审查
    ├── 信息披露要求
    ├── 各平台广告政策
    └── 监管约束

第 3-5 天：内容制作
├── 内容创作者 → 多格式内容制作
│   ├── 博客文章 / 长文
│   ├── 邮件序列
│   ├── 着陆页文案
│   ├── 视频脚本
│   └── 社交媒体文案（各平台定制版）
│
├── Twitter 互动官 → Twitter 专属内容
│   ├── 上线推文串（10-15 条）
│   ├── 每日互动推文
│   ├── 回复模板
│   └── 话题标签策略
│
├── TikTok 策略师 → TikTok 内容规划
│   ├── 视频方案（3-5 个视频）
│   ├── 钩子策略
│   ├── 热门音频/形式对齐
│   └── 发布排期
│
├── Instagram 策展人 → Instagram 内容
│   ├── 信息流帖子（轮播、单图）
│   ├── Stories 内容
│   ├── Reels 方案
│   └── 视觉风格规范
│
└── Reddit 社区建设者 → Reddit 策略
    ├── 目标 subreddit 选定
    ├── 价值优先的帖子草稿
    ├── 评论互动计划
    └── AMA 准备（如适用）
```

### 第 2 周：发布与激活

```
第 1 天：发布前准备
├── 所有内容已排队并定时
├── 数据追踪已验证
├── A/B 测试变体已配置
├── 着陆页已上线并测试
└── 团队已了解互动规则

第 2-3 天：发布
├── Twitter 互动官 → 发布推文串 + 实时互动
├── Instagram 策展人 → 发布帖子 + Stories
├── TikTok 策略师 → 发布视频
├── Reddit 社区建设者 → 社区真实发帖
├── 内容创作者 → 博客发布 + 邮件群发
├── 增长黑客 → 付费活动激活
└── 数据分析师 → 实时仪表盘监控

第 4-5 天：优化
├── 数据分析师 → 前 48 小时表现报告
├── 增长黑客 → 根据数据优化渠道
├── 实验追踪员 → A/B 测试早期结果
├── 社交媒体策略师 → 互动策略调整
└── 内容创作者 → 根据用户反应做新内容
```

### 第 3-4 周：持续与优化

```
每日：
├── 各平台智能体 → 互动和内容发布
├── 数据分析师 → 每日表现快照
└── 增长黑客 → 漏斗优化

每周：
├── 社交媒体策略师 → 活动表现复盘
├── 实验追踪员 → A/B 测试结果 + 新实验
├── 内容创作者 → 根据数据做新内容
└── 数据分析师 → 周度活动报告

活动结束时：
├── 数据分析师 → 全面活动分析
├── 增长黑客 → ROI 分析和渠道效果
├── 高管摘要生成器 → 活动高管摘要
└── 社交媒体策略师 → 经验总结和建议
```

## 活动指标

| 指标 | 目标 | 负责人 |
|------|------|--------|
| 总触达 | [根据预算定目标] | 社交媒体策略师 |
| 互动率 | 跨平台平均 > 3% | 各平台智能体 |
| 点击率 | CTA 点击率 > 2% | 增长黑客 |
| 转化率 | 着陆页转化 > 5% | 增长黑客 |
| 获客成本 | < [目标 CAC] | 增长黑客 |
| 品牌情感 | 正面净值 | 品牌守护者 |
| 内容产出量 | [目标数量] | 内容创作者 |
| A/B 测试完成数 | >= 5 | 实验追踪员 |

## 各平台 KPI

| 平台 | 主要 KPI | 次要 KPI | 智能体 |
|------|---------|---------|--------|
| Twitter/X | 曝光量 + 互动率 | 粉丝增长 | Twitter 互动官 |
| TikTok | 播放量 + 完播率 | 粉丝增长 | TikTok 策略师 |
| Instagram | 触达 + 收藏数 | 主页访问 | Instagram 策展人 |
| Reddit | 投票数 + 评论质量 | 引荐流量 | Reddit 社区建设者 |
| 邮件 | 打开率 + 点击率 | 退订率 | 内容创作者 |
| 博客 | 自然流量 + 页面停留 | 外链数 | 内容创作者 |
| 付费广告 | ROAS + CPA | 质量分数 | 增长黑客 |

## 品牌一致性检查点

| 检查点 | 什么时候 | 智能体 |
|--------|---------|--------|
| 发布前内容审核 | 每一条内容 | 品牌守护者 |
| 视觉一致性审计 | 每周 | 品牌守护者 |
| 语调检查 | 每周 | 品牌守护者 |
| 合规审查 | 发布前 + 每周 | 法务合规员 |
