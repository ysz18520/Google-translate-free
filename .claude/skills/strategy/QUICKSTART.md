# NEXUS 快速上手指南

> **从零到多智能体协作流水线，5 分钟搞定。**

---

## NEXUS 是什么？

**NEXUS**（Network of EXperts, Unified in Strategy，专家网络统一策略）把 Agency 的 AI 专家们变成一条协调的流水线。不再是一个个激活智能体、然后祈祷它们能配合，NEXUS 明确定义了谁在什么时候做什么、怎么验证质量。

## 选择你的模式

| 我想... | 使用 | 智能体数 | 时间 |
|--------|------|---------|------|
| 从零开始做一个完整产品 | **NEXUS-Full** | 全部 | 12-24 周 |
| 做一个功能或 MVP | **NEXUS-Sprint** | 15-25 | 2-6 周 |
| 搞一个具体任务（修 bug、搞活动、做审计） | **NEXUS-Micro** | 5-10 | 1-5 天 |

## NEXUS-Full：启动完整项目

**复制这个提示词来激活完整流水线：**

```
激活智能体编排者，NEXUS-Full 模式。

项目：[你的项目名称]
需求说明：[描述你的项目或链接到需求文档]

执行完整 NEXUS 流水线：
- 第 0 阶段：发现（趋势研究员、反馈分析师、UX 研究员、数据分析师、法务合规员、工具评估师）
- 第 1 阶段：策略（工作室制片人、高级项目经理、Sprint 排序师、UX 架构师、品牌守护者、后端架构师、财务追踪员）
- 第 2 阶段：基础搭建（DevOps 自动化师、前端开发者、后端架构师、UX 架构师、基础设施运维师）
- 第 3 阶段：构建（开发-测试循环 — 全部工程 + 证据收集者）
- 第 4 阶段：加固（现实检验者、性能基准师、API 测试员、法务合规员）
- 第 5 阶段：上线（增长黑客、内容创作者、全部营销智能体、DevOps 自动化师）
- 第 6 阶段：运营（数据分析师、基础设施运维师、客服响应者、持续进行）

每个阶段之间设质量门禁。所有评估必须有证据。
每个任务最多重试 3 次，超过就升级处理。
```

## NEXUS-Sprint：做一个功能或 MVP

**复制这个提示词：**

```
激活智能体编排者，NEXUS-Sprint 模式。

功能/MVP：[描述你要做的东西]
时间线：[目标周数]
跳过第 0 阶段（市场已验证）。

Sprint 团队：
- PM：高级项目经理、Sprint 排序师
- 设计：UX 架构师、品牌守护者
- 工程：前端开发者、后端架构师、DevOps 自动化师
- QA：证据收集者、现实检验者、API 测试员
- 支持：数据分析师

从第 1 阶段开始，做架构和 Sprint 规划。
所有实现任务跑开发-测试循环。
上线前必须拿到现实检验者的批准。
```

## NEXUS-Micro：做一个具体任务

**选你的场景，复制对应提示词：**

### 修一个 Bug
```
激活后端架构师，调查并修复 [BUG 描述]。
修完后激活 API 测试员验证修复。
然后激活证据收集者确认没有视觉回归。
```

### 跑一个营销活动
```
激活社交媒体策略师，担任 [活动描述] 的活动负责人。
团队：内容创作者、Twitter 互动官、Instagram 策展人、Reddit 社区建设者。
品牌守护者在发布前审核所有内容。
数据分析师每日跟踪效果。
增长黑客每周优化渠道。
```

### 做一次合规审计
```
激活法务合规员，做一次全面合规审计。
范围：[GDPR / CCPA / HIPAA / 全部]
审计完成后，激活高管摘要生成器输出利益相关方报告。
```

### 排查性能问题
```
激活性能基准师，诊断性能问题。
范围：[API 响应时间 / 页面加载 / 数据库查询 / 全部]
诊断完成后，激活基础设施运维师做优化。
DevOps 自动化师部署基础设施变更。
```

### 市场调研
```
激活趋势研究员，做 [领域] 的市场情报收集。
交付物：竞争格局、市场规模估算、趋势预测。
调研完成后，激活高管摘要生成器输出简报。
```

### UX 改进
```
激活 UX 研究员，找出 [功能/产品] 的可用性问题。
调研完成后，激活 UX 架构师设计改进方案。
前端开发者实现改动。
证据收集者验证改进效果。
```

## 核心概念 30 秒速览

1. **质量门禁** — 没有基于证据的批准，阶段不能推进
2. **开发-测试循环** — 每个任务做完就测；通过才继续，不通过就重试（最多 3 次）
3. **交接** — 智能体之间结构化的上下文传递（不能冷启动）
4. **现实检验者** — 最终质量权威；默认态度是"需要改进"
5. **智能体编排者** — 流水线控制器，管理整个流程
6. **证据高于口说** — 截图、测试结果和数据，不是口头声明
